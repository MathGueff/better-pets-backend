## Testaí

Testaí é um projeto de iniciativa para estudo, com foco em simular e gamificar o trabalho como QA.

### Estrutura do Projeto

- **/backend**: API Node.js/Express com TypeScript e Mongoose.
- **/frontend**: Aplicação React com Vite e Tailwind CSS.

### Tecnologias

- **Backend**: Node, Express, TypeScript, MongoDB, Mongoose, Zod, Multer, Cors, Dotenv.
- **Frontend**: React, TypeScript, Tailwind CSS, Axios.
- **CI/CD**: GitHub Actions, Semantic Release, DigitalOcean App Platform.
